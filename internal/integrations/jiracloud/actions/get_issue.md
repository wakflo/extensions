# Get Issue

## Description
Get details of a specific Jira issue.

## Properties
| Name      | Type   | Required | Description                            |
|-----------|--------|----------|----------------------------------------|
| projectId | string | Yes      | The project the issue belongs to       |
| issueId   | string | Yes      | The ID of the issue to retrieve        |

## Details
- **Type**: sdkcore.ActionTypeNormal
